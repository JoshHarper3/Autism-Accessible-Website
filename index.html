Mobile Guidelines
  1. Ensure website navigation is clearly labelled with clear language or appropriate use of icons that accurately represent the page and follows an intuitive structure.
  2. Avoid the use of pop-ups that interrupt user flow and focus and instead display that information within the relevant page of the website.
  3. Ensure page structure is clear with content being contained in the relevant page and section and not dispersed across different pages when unnecessary.
  4. Ensure buttons are clearly labelled with either a clear and appropriate icon or clear language to avoid user confusion.
  5. Ensure buttons are sufficiently large and are effectively spaced to minimise user mis-clicks.
  6. Ensure users can customise key visual and content features such as an adjustable colour scheme or the ability to condense text.
  7. Ensure videos do not play immediately on website launch, instead a video must require user input to play.
  8. Ensure there is adequate spacing of all website elements and adapt layout from desktop if required to ensure the website is not too visually cluttered.
